Map Distortions for the Statistics Canada Lambert (EPSG:3347)

The Statistics Canada Lambert (EPSG:3347) is a map projection widely used for mapping the entirety of Canada. The interactive map below illustrates the impact of varying standard parallels on the map's distortions through the display of Tissot's Indicatrices at set intervals. It's important to note that in conformal map projections like this one, Tissot's ellipses alter in size but maintain their shape, meaning the scale factors, h and k, are identical. Adjacent to the map, a table presents the h and k values alongside a dynamic bar graph for easy comparison. The source code can be reviewed here.
